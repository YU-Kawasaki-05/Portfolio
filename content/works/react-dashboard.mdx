# React Analytics Dashboard

データドリブンな意思決定を支援するリアルタイム分析ダッシュボードです。

## プロジェクト概要

- **目的**: 複雑なデータを直感的に理解できるUI構築
- **期間**: 2024年9月〜11月（3ヶ月）
- **役割**: フロントエンド開発・UX設計

## 主要機能

### リアルタイムデータ
- WebSocket接続による即座更新
- 1秒間隔でのメトリクス反映
- 接続状態の視覚的フィードバック

### インタラクティブチャート
- D3.js による カスタムビジュアライゼーション
- ズーム・パン操作対応
- データポイントの詳細表示

### フィルタリング機能
- 期間・カテゴリ・地域別絞り込み
- URLパラメータでの状態保持
- エクスポート機能（CSV・PNG）

## 技術的成果

- **パフォーマンス**: 10,000データポイントでも60fps維持
- **アクセシビリティ**: WCAG 2.1 AA準拠
- **レスポンシブ**: モバイルファーストデザイン
