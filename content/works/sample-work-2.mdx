# Interactive Portfolio Dashboard

Next.js 15とFramer Motionを使用したモダンなポートフォリオダッシュボードです。

## プロジェクト概要

このプロジェクトは、従来の静的なポートフォリオサイトを超えて、インタラクティブなダッシュボード形式でスキルや実績を可視化するWebアプリケーションです。

### 主な機能

- **動的データ可視化**: チャートとグラフによるスキル表示
- **スムーズアニメーション**: Framer Motionによる滑らかな画面遷移
- **レスポンシブレイアウト**: モバイルファーストなデザイン
- **ダークモード対応**: システム設定に応じた自動切り替え

### 技術スタック

- **Framework**: Next.js 15, React 19
- **Animation**: Framer Motion
- **Styling**: Tailwind CSS
- **Charts**: Chart.js, D3.js
- **Deployment**: Vercel

## 開発の特徴

### パフォーマンス最適化

- **Image Optimization**: Next.js Image コンポーネントによる自動最適化
- **Code Splitting**: 動的インポートによる初期読み込み時間の短縮
- **Lazy Loading**: スクロール位置に応じたコンテンツの遅延読み込み

### アクセシビリティ

- **キーボードナビゲーション**: 全機能のキーボード操作対応
- **スクリーンリーダー対応**: ARIA属性による適切な情報提供
- **カラーコントラスト**: WCAG 2.1 AA準拠の色彩設計

## 成果

- **Lighthouse Score**: Performance 98, Accessibility 100
- **Core Web Vitals**: LCP < 1.5s, FID < 100ms, CLS < 0.1
- **ユーザビリティ**: 直感的な操作性と高い満足度を実現
