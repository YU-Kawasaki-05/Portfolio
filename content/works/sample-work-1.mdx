# Neo-Typographic Web Application

React + Three.jsを使用したインタラクティブなタイポグラフィWebアプリケーションです。

## プロジェクト概要

このプロジェクトは、従来の2Dタイポグラフィの概念を3D空間に拡張し、ユーザーとのインタラクションによって動的に変化するテキスト表現を実現しました。

### 主な機能

- **3Dタイポグラフィ**: Three.jsを使用した立体的な文字表現
- **インタラクティブアニメーション**: マウス操作に応じたリアルタイム変形
- **レスポンシブデザイン**: デバイスサイズに応じた最適化
- **パフォーマンス最適化**: WebGLを活用した高速レンダリング

### 技術スタック

- **Frontend**: React 18, TypeScript
- **3D Graphics**: Three.js, React Three Fiber
- **Animation**: Framer Motion, GSAP
- **Styling**: Tailwind CSS
- **Build Tool**: Vite

## 開発プロセス

### 1. コンセプト設計

タイポグラフィとテクノロジーの融合をテーマに、文字が持つ表現力を3D空間で最大化することを目標としました。

### 2. プロトタイピング

Three.jsの基本機能を使用して、文字の3D化とアニメーション効果の検証を行いました。

### 3. 実装

React Three Fiberを使用してReactコンポーネントとして3Dシーンを構築し、状態管理とアニメーション制御を統合しました。

## 成果と学び

- **技術的成果**: WebGLを活用した高性能な3Dタイポグラフィシステムの構築
- **デザイン成果**: 従来のWebデザインの枠を超えた表現手法の確立
- **学習成果**: 3Dグラフィックスとフロントエンド開発の融合技術の習得

このプロジェクトを通じて、Webブラウザ上での3Dグラフィックス表現の可能性と、ユーザーエクスペリエンスの新しい形を探求することができました。
